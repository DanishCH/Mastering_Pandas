# 🐼 Pandas Practice From Scratch

Hands-on practice notebooks for learning **Pandas**, one of Python's most
powerful libraries for data analysis. All notebooks are created and run in
**Google Colab**.

##  What's Covered

- Creating Series and DataFrames
- Reading data (CSV, Excel)
- Selecting, filtering, and sorting data
- Handling missing values and data cleaning
- GroupBy and aggregation
- Merging and joining DataFrames
- Basic data visualization

##  Tools Used

- Python 3
- Pandas
- NumPy
- Google Colab

##  How to Use

Open any notebook and click the **"Open in Colab"** badge at the top to run
it directly in your browser — no installation needed.

##  Purpose

This repository documents my journey learning data analysis with Python as
part of my studies, including the **AI Using Python** course (DigiSkills 3.0).
New notebooks are added as I practice more topics.
